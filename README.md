# Polymarket-Quant

这是一个面向 Polymarket 等预测市场的量化套利研究项目。首轮实现只搭建全六层脚手架，并用离线样例数据验证 P1 数据与信号闭环。

当前阶段边界：
- 只读，不下单，不签名，不上链。
- 验证策略①单条件结构套利与策略④延时套利信号。
- 洗盘检测与出场流动性预检是信号前置过滤。
- 离线 replay 只验证数据口径和信号规则，不宣称真实收益、夏普或可实盘。

本机当前目标运行时是 Python 3.14。文档全量依赖已写入 `pyproject.toml`，如果 OpenBB、web3、mcp 等包在 Python 3.14 下不兼容，安装阶段应保留失败证据并停止，不自动切换版本。

验证命令：

```powershell
python -m pytest
```
